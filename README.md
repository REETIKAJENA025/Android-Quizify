# QUIZIFY -- Learn, Play and Repeat

QUIZIFY is an interactive Android quiz application designed to make
learning fun, dynamic, and engaging. It offers multiple quiz categories,
randomized questions, timers, instant scoring, and personalized feedback
to improve user learning and retention.

## Short Description

A dynamic Android quiz app featuring category-based quizzes, random
questions, real-time scoring, and an interactive user experience.

## Features

### Multiple Quiz Categories

Users can choose from diverse topics including: - General Knowledge -
Mathematics - Science - Technology - History

### Randomized Questions

-   Each quiz session presents 7 random questions out of a larger
    question bank.
-   Fisher--Yates shuffling ensures unbiased question and option
    randomization.
-   Promotes replayability and prevents memorization.

### Built-in Timer

-   Countdown timer adds challenge and time awareness.
-   Encourages quick thinking and improves decision-making speed.

### Instant Feedback

-   Users get immediate correct/incorrect responses.
-   Enhances learning through real-time correction.
-   Motivates continuous improvement.

### Scoring System

-   Final score out of 7 displayed at the end.
-   Real-time updates after each question.
-   Personalized comments based on performance.

### Result Screen

-   Displays final score with animations and encouraging messages.
-   Includes a "Play Again" option for a fresh quiz.

## Technology Used

-   Android Studio (Java)
-   Activities & Intents
-   CountDownTimer
-   Randomization algorithms
-   XML-based UI designs

## How It Works

1.  User selects a quiz category.
2.  Randomized questions are displayed.
3.  Timer runs for each question.
4.  Instant feedback is provided.
5.  Final score and remarks are shown.
6.  User can replay with newly shuffled questions.
